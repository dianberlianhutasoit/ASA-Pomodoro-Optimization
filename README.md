# Optimasi Alokasi Waktu Belajar Mandiri Mahasiswa Berbasis Teknik Pomodoro dengan Batasan Kapasitas Fokus Menggunakan Algoritma Greedy, Dynamic Programming, dan Simulated Annealing

Repository ini berisi kode program untuk eksperimen perbandingan Greedy Algorithm, Dynamic Programming, dan Simulated Annealing pada kasus optimasi alokasi aktivitas belajar mandiri mahasiswa berbasis teknik Pomodoro.

## Struktur File

- `main.py` : file utama untuk menjalankan eksperimen.
- `greedy_algorithm.py` : implementasi Greedy Algorithm.
- `dynamic_programming.py` : implementasi Dynamic Programming.
- `simulated_annealing.py` : implementasi Simulated Annealing.
- `Lampiran Dataset ASA.xlsx` : dataset aktivitas belajar mahasiswa.
